About this APP

This is the final application made for the course Dynamic Web Applications in Laurea University of Applied Sciences.

How to use this app?
Simply select the theater from the dropdown and then all the showings will be displayed in a table. The application shows only current day´s shows that are still available. So if you search for shows at 17:00, it will only display shows after that time.
After you have picked a theatre, you can filter your search by typing the movie you want to watch.
Known bugs...
  1. This application relies on the API provided by Finnkino. I have developed this app since 13th of December and as far as I know there have been no problems thus far.
  2. Finnkino apparently changes the schedules every tuesday and on tuesdays there might be some minor problems with the API, but that remains to be seen...
  3. Edit: Apparently if you filter the movies with the search input, it will also hide the navigation links. I noticed this too late and wasn´t able to fix the bug.
