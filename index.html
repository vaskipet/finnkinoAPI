About this APP

This is the final application made for the course Dynamic Web Applications in Laurea University of Applied Sciences.

How to use this app?
  1. Simply select the theater from the dropdown and then all the showings will be displayed in a table.
  2. The application shows only current dayÂ´s shows that are still available. So if you search for shows at 17:00, it will only display shows after that time.
  3. After you have picked a theatre, you can filter your search by typing the movie you want to watch.
  4. You can hide the search bar from the button under the jumbotron to see a bit better the results.
Known bugs...
  1. This application relies on the API provided by Finnkino. I have developed this app since 13th of December and as far as I know there have been no problems thus far.
  2. Finnkino apparently changes the schedules every tuesday and on tuesdays there might be some minor problems with the API, but that remains to be seen...
  3. Also during the time when the API is being refreshed (the day is changing or there are new shows being scheduled), there might be some bugs with pictures and booking-links. It will take about couple of hours before the problem is fixed.
    • =>Because the problem is only temporary, there is yet no error-handling solution to it.
